# Chrome Web Store listing — WebMakeover v1.0.0

Copy/paste fields for the Developer Dashboard.

## Name
WebMakeover — Reading Studio

## Summary (132 chars max)
Make any web page easy to read. Tune zoom, text size, spacing and color, or pick a reading preset. Settings saved per site.

## Description
WebMakeover turns any web page into a comfortable reading surface.

Open the popup and shape the page in real time:

• Zoom the whole page from 50% to 200%
• Scale text relative to its original size, so headings and body keep their proportions
• Open up line spacing for easier reading
• Set your own background (paper) and text (ink) colors
• One‑click presets: Reading, Dark, Sepia and High‑Contrast
• A live preview shows the effect before it touches the page
• Your settings are remembered per website and re‑applied automatically
• One button restores the original page at any time

Everything runs locally in your browser. WebMakeover does not collect, store, or
transmit any data. It has no servers and makes no network requests.

Great for reducing eye strain, reading long articles, low‑vision accessibility,
or just making a cluttered site pleasant to read.

## Category
Accessibility

## Single purpose
Let users adjust the visual appearance (zoom, text size, spacing, colors) of the
current web page for more comfortable reading.

## Permission justifications
- **activeTab**: Apply the user's chosen appearance to the page they are viewing,
  only when they open the popup and act.
- **scripting**: Inject the styling (font scaling, spacing, colors, zoom) into the
  active tab and cleanly remove it on reset.
- **storage**: Remember the user's per‑site settings locally so the page looks the
  same next visit. Nothing leaves the device.

## Data usage disclosures
- Does the extension collect user data? **No.**
- All settings are stored locally via chrome.storage and never transmitted.

## Privacy policy
Not required (no data collected). If one is requested, a one‑line statement:
"WebMakeover stores your display preferences locally on your device and collects
no personal data."
